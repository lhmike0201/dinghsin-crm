# 鼎新 ERP 客戶需求蒐集平台

單頁式業務工具，供業務員在客戶現場快速蒐集需求資訊、自動對應鼎新產品組合，並匯出給後續解決方案／簡報流程使用。

## 功能

- **智能辨識填入**：貼會議筆記自動辨識 13+ 欄位
- **商機等級 A/B/C1/C2/D**：色卡選擇器
- **多客戶管理**：搜尋、篩選、排序、編輯／複製／刪除
- **草稿自動存檔**：localStorage 每 3 秒
- **匯出格式**：JSON（第二階段對應用）、Markdown（簡報用）、複製結構化文字、列印
- **初步解決方案建議**：依通路／痛點／規模／現行系統自動推薦鼎新產品
- **RWD**：手機、平板、桌機都能用

## 技術

- 純 HTML/CSS/JS 單檔（`index.html`）
- 無外部依賴、無後端、資料存在瀏覽器 localStorage

## 使用

直接開啟 `index.html` 或到部署後的網址即可使用。
